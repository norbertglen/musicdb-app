## Running The Project

Clone the project, then run:

### `cd [path-to-clone]`

### `npm install`

### `npm start`

This runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.

Project built on the following technologies:
1. React
2. Typescript

Node version: v10.21.0
